Pesquisa de Satisfação de Usuários | Solar BPM

Olá,

A Softplan promove, três vezes ao ano, uma pesquisa de satisfação com os usuários do Solar BPM e, no próximo dia 10/03, lançará o 1º ciclo.

1º ciclo | 10 de março a 25 de abril
A pesquisa de usuários é composta por uma pergunta inicial e algumas opções de classificação, bem como campo para comentários. Ela aparecerá diretamente no sistema quando o usuário estiver logado.


Seu feedback será ouvido e valorizado e retornará em formato de uma melhor experiência de uso.
Participe!
A voz de quem utiliza o sistema é essencial para a evolução das nossas soluções.
